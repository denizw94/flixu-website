# Flixu Copywriting & Brand Voice Rule

> [!IMPORTANT]  
> This rule applies whenever you write, edit, or evaluate any marketing copy, blog posts, UI text, or documentation for Flixu.

## 1. Core Identity
- **Voice:** Calm, Precise, Warm, Human. 
- **The Cardinal Sin:** Never make the copy about "how great Flixu is". Always make it 100% about the *user's outcome* and their business problem.

## 2. Terminology & Tone
- **NO HYPE:** Never use words like "revolutionary", "game-changing", "crushing it", "magic", or "best-in-class".
- **Clarity Beats Cleverness:** Write at a 6th-grade reading level. Use simple, everyday words. Avoid academic or overly intellectual phrasing.
- **Specifics Over Generics:** Replace vague claims ("incredible ROI") with concrete data ("40% faster QA cycles").
- **Forum Foraging:** Use the exact vernacular of the target audience (e.g., translators, localization managers) instead of inventing marketing jargon.

## 3. Structural Requirements
- **Answer-First:** Always place the TL;DR, definition, or primary value proposition in the first 100 words.
- **Compact Clarity:** Keep paragraphs between 60–100 words. Stick to one core idea per paragraph.
- **Punchy Sentences:** Keep sentences between 15–20 words on average. 
- **Sell the Click:** Ensure every section, paragraph, and CTA has exactly *one* job. If it's an ad, it sells the click. If it's a landing page, it sells the trial.
- **PAS Framework:** When addressing top-of-funnel (Stage 2) users, always use the Problem -> Agitation -> Solution structure before introducing Flixu features.

## 4. Dos and Don'ts
- **DO:** Be honest about limitations and respectful when comparing competitors ("DeepL is excellent at raw speed. Flixu focuses on enterprise consistency.").
- **DON'T:** Use generic filler sentences ("In today's fast-paced digital landscape...").
- **DON'T:** Use the "helicopter perspective" in storytelling. Zoom into the moment (Location, Action, Emotion, Dialogue) to make it real.
